1. Download the entire folder containing all required files and scripts to your computer. The application will not work without these files, except for the sample_20.csv file.
2. Launch desktopApp.py (if it doesn't open with a double-click, right-click on the file and select Open with -> Python).
3. Select the data file (a sample file sample_20.csv is included in the folder for testing purposes).
4. In the interface, click on Process File.
5. If the program runs successfully, all four statuses (First script, Second script, Third script, Fourth script) will display Completed.
6. The interface will show the location and name of the result file.
